# File Server

В этой работе я запустил контейнер для файлового обмена и проверил доступ к файлам через браузер.
Этот контейнер поднимает простой файловый сервер, через который можно открыть содержимое папки и скачать нужные файлы.

## Команда запуска

```powershell
docker run -d --name file-server -p 8084:80 -v ${PWD}:/srv halverneus/static-file-server:latest
```

![Запуск файлового сервера](img/01-run.png)

## Проверка

```powershell
docker ps
```

После запуска я открыл адрес http://localhost:8084 и убедился, что сервер показывает содержимое папки.

![Содержимое папки в браузере](img/02-browser.png)
